# Example GIE (nvinfer) config — YOLO detection.
#
# Pass this to osprey.configure(gie_config="…/gie.txt").
#
# THREE paths below are environment-specific — set them before use:
#   onnx-file / model-engine-file  -> your model
#   labelfile-path                 -> your class labels (one per line)
#   custom-lib-path                -> the parser .so that SHIPS WITH ospreyai.
#
# Resolve the shipped parser path with:
#   python3 -c "import osprey,os;print(os.path.join(os.path.dirname(osprey.__file__),'server','deepstream','lib','nvdsinfer_yolo_det.so'))"
#
# Parsers shipped in the package (osprey/server/deepstream/lib/):
#   nvdsinfer_yolo_det.so  -> YOLO detection      (parse-bbox-func-name=NvDsInferParseCustomYoloDet)
#   nvdsinfer_yolo_seg.so  -> YOLO segmentation
#   nvdsinfer_rtdetr.so    -> RT-DETR detection
#
# Do NOT point custom-lib-path at /deepstream_app/… — that was the old dev
# layout; the parser now lives inside the installed ospreyai package.
